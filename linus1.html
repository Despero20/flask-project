    LINUX COMMANDS

    pwd-- print name of current/working directory

    man-- shows a manual page for a certain topic

    ls-- shows the folder you are in

    /-- show all the files and directories in the root directory

    ..-- it takes you to the previous directory

    echo-- to display

    tail filename-- for end of file

    cd-- change directory

    cd..-- to parent directory

    cat-- it reads from the standard output to input

    | >-- it change the standard output

    whoami-- brings out the standard output
